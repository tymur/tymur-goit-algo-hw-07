# Дерева та балансування: Завдання 1–4

Цей проєкт демонструє реалізацію базових структур даних та алгоритмів для роботи з деревами (BST, AVL) та ієрархічними структурами (дерево коментарів).

---

## Завдання 1: Пошук найбільшого значення в дереві
- **Що зроблено:**  
  Реалізовано алгоритм, який знаходить найбільше значення у двійковому дереві пошуку (BST) або AVL-дереві.  
- **Як працює:**  
  Алгоритм іде по дереву праворуч, поки не досягає вузла без правого нащадка.

---

## Завдання 2: Пошук найменшого значення в дереві
- **Що зроблено:**  
  Реалізовано алгоритм для знаходження найменшого значення у BST або AVL-дереві.  
- **Як працює:**  
  Алгоритм іде по дереву ліворуч, поки не досягає вузла без лівого нащадка.

---

## Завдання 3: Підрахунок суми значень у дереві
- **Що зроблено:**  
  Створено рекурсивний алгоритм для обчислення суми всіх значень у BST або AVL-дереві.  
- **Як працює:**  
  Для кожного вузла рекурсивно додає значення лівого піддерева, значення вузла та правого піддерева.

---

## Завдання 4: Ієрархічна система коментарів
- **Що зроблено:**  
  Реалізовано клас `Comment` для представлення системи коментарів з можливістю додавати відповіді (формування дерева коментарів).  
- **Додатково:**  
  - Метод `add_reply` додає відповідь до коментаря.  
  - Метод `remove_reply` позначає коментар як видалений.  
  - Метод `display` рекурсивно виводить дерево коментарів із відступами.  
  - Додана **візуалізація дерева коментарів** у вигляді графа за допомогою `networkx`.

---

- `networkx` та `matplotlib` для візуалізації (у завданні 4)
